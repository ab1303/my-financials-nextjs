# Finance Chat Assistant — Context

## Problem Statement
Users want to query their financial data (transactions, cashflow, budgets) in natural language. The system should provide an LLM-powered chat interface that understands finance queries and returns structured, actionable responses.

## Domain Dependencies
- AI Features HLD (../hld.md)
- Transactions, Cashflow, Budgets
- AI model integration (OpenAI, GitHub)

## Scope
- LLM-based chat interface for finance
- Prompt engineering for accurate, safe responses
- (Future) Retrieval-augmented generation (RAG) for up-to-date data
- Track and log all AI usage
